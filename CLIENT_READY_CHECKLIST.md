# Чек-Лист Перед Сдачей Клиенту

## 1) Локальная Проверка

- [ ] Команда `streamlit run web_app.py` запускается без ошибок.
- [ ] Открывается `http://127.0.0.1:8501`.
- [ ] Режим `Текст статьи` выдает балл, проблемы и план улучшений.
- [ ] Режим `URL страницы` успешно анализирует хотя бы 1 сайт.
- [ ] Кнопка `Скачать отчет (.md)` скачивает файл.

## 2) Проверка AI

- [ ] Задан `OPENAI_API_KEY`.
- [ ] Работает генерация `AI-рекомендаций`.
- [ ] Работает генерация `AI-вариантов улучшенного текста`.

## 3) Публичный Деплой

- [ ] Проект загружен в GitHub без `.venv` и секретов.
- [ ] В Streamlit Cloud выбран файл `web_app.py`.
- [ ] В Secrets добавлен `OPENAI_API_KEY`.
- [ ] Публичная ссылка открывается с другого устройства.

## 4) Что Передать Клиенту

- [ ] Публичная ссылка на приложение.
- [ ] Краткая инструкция использования (2-3 шага).
- [ ] Ограничения: анализ URL зависит от сетевых настроек и SSL/прокси.
