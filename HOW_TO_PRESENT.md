# Как Защитить Проект

## Короткий питч (30 секунд)

Это веб-инструмент для SEO-аудита контента. Он принимает URL страницы или текст статьи, оценивает SEO-качество по набору правил, показывает приоритетные проблемы и дает понятный план исправлений. При необходимости подключается ИИ для генерации дополнительных рекомендаций и улучшенных вариантов текста.

## Что показать на демо

1. Вставить URL страницы и нажать анализ.
2. Показать итоговый балл и список проблем с приоритетами.
3. Открыть блок метрик и объяснить 2-3 ключевых показателя.
4. Нажать "Скачать отчет (.md)" и показать готовый результат для команды.
5. Включить AI-опции и показать улучшенные варианты текста.

## Архитектура

- `web_app.py`: веб-интерфейс на Streamlit.
- `seo_ai_analyzer/analyzer.py`: правила SEO-анализа и расчет балла.
- `seo_ai_analyzer/fetcher.py`: загрузка HTML по URL.
- `seo_ai_analyzer/ai_layer.py`: интеграция с OpenAI для рекомендаций и переписывания текста.
- `seo_ai_analyzer/cli.py`: консольный режим запуска.

## Почему это полезно

- Быстрый аудит контента до публикации.
- Понятные рекомендации вместо "сырых" метрик.
- Единый процесс для редактора, маркетолога и SEO-специалиста.

## Честные ограничения

- Правила оценки упрощенные и не заменяют глубокий технический аудит.
- Для AI-функций нужен `OPENAI_API_KEY`.
- Качество рекомендаций зависит от качества исходного текста и выбранной модели.
