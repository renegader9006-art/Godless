# Godless
Проект выполняли: **faza** и **Fals043k**.

## О Проекте

Мы сделали этот инструмент как практичный MVP для анализа SEO-контента без сложной настройки.
Наша цель: чтобы можно было быстро проверить страницу или статью, увидеть проблемы и сразу получить понятный план улучшений.

Что умеет:
- принимает URL страницы или текст статьи;
- анализирует SEO-показатели;
- показывает проблемы с приоритетами;
- дает понятные рекомендации по улучшению;
- опционально генерирует AI-рекомендации и 2 улучшенных варианта текста.

## Наши Решения

- Выбрали `Streamlit`, чтобы быстро запустить удобный веб-интерфейс.
- Разделили проект на модули: `analyzer`, `fetcher`, `ai_layer`, `cli`.
- Добавили два режима: анализ URL-страницы и анализ текста статьи.
- Сделали русскоязычный вывод и экспорт отчета в `.md`.

- ## Что Оценивается

- длина `title` и `meta description`;
- структура `H1/H2`;
- глубина контента (объем текста);
- покрытие `alt` у изображений;
- внутренние и внешние ссылки;
- `canonical` и OpenGraph-метаданные;
- плотность ключевых слов;
- читаемость текста (в режиме статьи).
